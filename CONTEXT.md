# FridgeChef

แอป PWA แบบ Offline-First สำหรับสุ่มเมนูอาหารไทยจากวัตถุดิบที่ผู้ใช้มีในตู้เย็น — matching และ random ทำงานในเครื่องทั้งหมด

## Language

**Recipe (เมนู)**:
หนึ่งจานที่สุ่มได้ ประกอบด้วยวัตถุดิบหลัก เครื่องปรุง ขั้นตอนทำ และข้อมูลประกอบ (เวลา/ความยาก/จำนวนที่) — ใน UI ภาษาไทยแสดงคำว่า "เมนู" ได้ แต่ในโค้ดและ DB ใช้ Recipe เสมอ
_Avoid_: Menu, Dish, Food

**Ingredient (วัตถุดิบ)**:
สิ่งที่ผู้ใช้เลือกว่ามีในตู้เย็น และเมนูใช้อ้างอิงเพื่อคิดคะแนนความเข้ากัน มีตัวตนถาวรเป็น slug id (เช่น `pork-minced`) คู่กับชื่อไทยสำหรับแสดงผล — "หมู" กับ "หมูสับ" เป็นคนละ Ingredient
_Avoid_: item, ของ, เครื่องปรุง

**Seasoning (เครื่องปรุง)**:
ของปรุงรสที่ถือว่าครัวมีอยู่แล้ว ผูกกับเมนูพร้อมปริมาณ (เช่น น้ำปลา 1 ชช.) แสดงในหน้าวิธีทำเท่านั้น **ไม่นำมาคิดคะแนน matching**
_Avoid_: ingredient, ส่วนผสม

**Core Ingredient (วัตถุดิบหลัก)**:
Ingredient ที่ผูกกับ Recipe และถูกนำมาคิดคะแนนความเข้ากัน (coverage) — คือเส้นความสัมพันธ์ Recipe↔Ingredient หนึ่งเส้น

**Required Ingredient (วัตถุดิบบังคับ)**:
Core Ingredient ที่ติด flag ว่าขาดไม่ได้ — ถ้าผู้ใช้ไม่มี Recipe นั้นจะถูกกรองทิ้งทั้งจาน (เช่น ใบกะเพราของผัดกะเพรา)
_Avoid_: requiredTags (ชื่อเดิมในสเปก — มันไม่ใช่ tag)

**Protein (โปรตีน)**:
Core Ingredient ที่ติด flag ว่าเป็นเนื้อสัตว์/โปรตีนหลักของจาน ใช้ให้คะแนนโบนัสตอน matching — Ingredient ตัวเดียวเป็นทั้ง Required และ Protein ได้ (เช่น กุ้งในต้มยำกุ้ง)

**Match Score (คะแนนความเข้ากัน)**:
ตัวเลขบอกว่า Recipe เข้ากับของใน Fridge แค่ไหน — ยิ่งมี Core Ingredient ครบยิ่งสูง มีโปรตีนตรงได้โบนัส ขาดมากโดนหัก
_Avoid_: rating, rank

**Coverage**:
สัดส่วน Core Ingredient ที่ผู้ใช้มีต่อทั้งหมดของ Recipe (0–1) — ต่ำกว่าเกณฑ์ขั้นต่ำถือว่าไม่เข้ารอบ

**Candidate (เมนูเข้ารอบ)**:
Recipe ที่ผ่านการกรอง (มีวัตถุดิบบังคับครบ, coverage ถึงเกณฑ์) และติดกลุ่มคะแนนสูงสุด เป็นตัวตั้งต้นของการสุ่มและการสลับเมนู — ใน UI เรียก "ตัวเลือกสำรอง"

**Fridge (ตู้เย็น)**:
ชุด Ingredient ที่ผู้ใช้ระบุว่ามีอยู่ตอนนี้ เก็บถาวรในเครื่องของผู้ใช้เท่านั้น (ไม่ผูกกับ server) และคงอยู่ข้ามการปิดเปิดแอป
_Avoid_: selection, cart

**Tag (ป้ายกำกับ)**:
ป้ายอิสระของ Recipe เพื่อบรรยายลักษณะจาน เช่น เผ็ด, จานด่วน, เด็กกินได้ — **ไม่ใช่**การอ้างอิงวัตถุดิบ และไม่มีผลต่อคะแนน matching
_Avoid_: ใช้คำว่า tag กับวัตถุดิบบังคับ
